# Localhost Address Matching

This project captures matching, normalization, and parsing of an address string.

## About the Project
This project was developed for the **TEKNOFEST 2025 Hepsiburada AI Hackathon**.

## Methodology
We have used a standard parser and a custom NER-based model trained with synthetic data by grouping data categories.
